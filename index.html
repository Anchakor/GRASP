GRASP

A visual RDF & ontology proof-of-concept editor

Download this project as a .zip file Download this project as a tar.gz file

Developed as a part of bachelor thesis Visual Design of Ontologies for Semantic Web which contains description and documentation of the editor.

Dependencies

  • Raptor >= 2.0.7
  • Redland with Berkley DB support (tested 1.0.13)
  • Rasqal (tested 0.9.25)
  • Qt with QtSvg (tested 4.7)
