GRASP - ontology/RDF editor

Developed as a part of bachelor's thesis Visual Design of Ontologies for Semantic Web which contains description and documentation of the editor.

Dependencies

  • Raptor >= 2.0.7
  • Redland with Berkley DB support (tested 1.0.13)
  • Rasqal (tested 0.9.25)
  • Qt with QtSvg (tested 4.7)
